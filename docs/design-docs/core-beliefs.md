# Core Beliefs

## Belief 1

Raw notes are the source of truth and must remain untouched.

## Belief 2

Generated pages are useful because they are readable, not because they replace raw evidence.

## Belief 3

Local-first is a product constraint, not a branding line.

## Belief 4

MVP should solve source-summary generation well before attempting concept synthesis.

## Belief 5

User trust depends more on provenance and safety than on model cleverness.
